# 🎨 LazySVG — Интерактивный редактор SVG
[Read it in English 🇬🇧](README.md)

Простой веб-инструмент для создания векторных фигур с поддержкой кривых Безье.

## ✨ Возможности

- **Режим линий** - Создание фигур из прямых линий
- **Режим Безье** - Создание плавных кривых с автоматическими контрольными точками
- **Визуализация** - Видимые точки и контрольные точки на холсте
- **Сетка** - Фоновая сетка для удобного позиционирования
- **Экспорт** - Копирование SVG и HTML-кода для использования в проектах
- **Интерактивный холст** - Добавление точек простым щелчком

## 🚀 Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev
```

## 📖 Как использовать

### Добавление точек
1. Выберите режим (Линия или Безье) слева
2. Щелкните по белой области холста, чтобы добавить точку
3. Точки будут соединены линиями или кривыми в зависимости от выбранного режима

### Режим линий
- Создает четкие линии углы между точками
- Идеально подходит для простых геометрических фигур

### Режим Безье
- Автоматически создает контрольные точки для плавных кривых
- Визуально отображает контрольные точки (синие и зеленые круги)
- Лучше всего подходит для органических форм и плавных кривых

### Экспорт
1. Скопируйте код SVG из раздела "Код SVG"
2. Используйте HTML-код из раздела "HTML-код" или вставьте SVG в свой проект

## 📝 Примеры использования

### Экспортированный SVG-код

Вы можете скопировать SVG-код и вставить его:

```html
<svg viewBox="0 0 600 400" xmlns="http://www.w3.org/2000/svg">
  <path d="M" 152 252 C 212 252 192 102 252 102 ..."
  fill="none" stroke="black" stroke-width="2"/>
</svg>
```

### Полная HTML-страница

Экспорт HTML-файла со встроенным SVG и стилями для быстрого использования.
